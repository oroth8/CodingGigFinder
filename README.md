# Coding Gig Finder
![Heroku Link](https://thawing-brook-03881.herokuapp.com/)

An app that utilizes SQL to help find stored coding gigs on a SQL server.

## Installation

Use the package manager Node.js to install Coding Gig Finder's dependencies.

```bash
npm i
```

## Usage

```python
npm start
```
